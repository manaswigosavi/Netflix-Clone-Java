# Netflix-Clone-Java
A Netflix-inspired web application developed using Core Java, JSP, Servlets, JDBC, MySQL, HTML, CSS, and Apache Tomcat.

## Features
- User Registration
- User Login
- Secure Session Management
- Logout Functionality

## Technologies Used
- Core Java
- JSP & Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat

## Prerequisites
- Java JDK 8+
- Eclipse IDE
- Apache Tomcat 9
- MySQL Server

## How to Run
1. Import the project into Eclipse.
2. Configure Apache Tomcat 9.
3. Create the required MySQL database.
4. Update the database password in DBConnect.java.
5. Run the project on Tomcat.
6. Open http://localhost:8080/NetflixClone/.
